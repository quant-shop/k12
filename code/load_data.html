Introduction

School STEM and academic supports—such as after-school STEM programs, mentoring, field trips, credit recovery, homework help, and dropout prevention—are unevenly distributed across U.S. schools. These supports collectively shape students’ opportunities to learn, particularly in mathematics and science. Yet most quantitative work treats these supports as separate variables rather than as bundled “regimes” of opportunity, potentially obscuring patterned inequities in how supports co-occur across schools.

In this study, we use latent class analysis (LCA) to identify distinct school-level regimes of STEM enrichment and academic support. We ask how schools cluster into qualitatively different configurations of supports, and which kinds of schools are more or less likely to provide comprehensive regimes versus minimal or remediation-only regimes. We situate this work in opportunity-to-learn theory, multi-tiered systems of support, and equity-focused quantitative traditions that emphasize structural patterns rather than individual deficits.

Research Questions and Hypotheses

Research Questions

  • RQ1: How do U.S. schools cluster into distinct regimes of STEM enrichment and academic support based on their provision of after-school STEM, mentoring, STEM-related field trips, credit recovery, homework help, and dropout prevention?
  • RQ2: How are school characteristics (e.g., sector/control, locale, student socioeconomic composition) associated with membership in each support regime?
  • RQ3 (optional, if you have student outcomes later): How are these support regimes associated with students’ STEM aspirations or achievement, net of student background?

Hypotheses

  • H1 (regime structure): Schools will sort into at least three distinct regimes:
    1. a high-support / comprehensive regime with high probabilities of both STEM enrichment and academic safety nets;
    2. a basic STEM-only, no-remediation regime with some enrichment but limited remediation; and
    3. a remediation-focused, weak-STEM-enrichment regime emphasizing credit recovery with fewer STEM opportunities.
  • H2 (school characteristics): Comprehensive support regimes will be more common in better-resourced schools (e.g., lower poverty, suburban or non-rural locales), while minimal or remediation-only regimes will be more prevalent in high-poverty and minoritized-serving schools.
  • H3 (if using outcomes): Students in comprehensive support regimes will show higher STEM aspirations or achievement than similar students in minimal or remediation-only regimes.

Analytic approach

Latent class analysis

  • Use latent class analysis (LCA) to identify qualitatively distinct regimes based on the six binary support indicators.
  • Specify that models with K = 2–4 classes are estimated using the poLCA package in R, with multiple random starts (e.g., 20) and a high maximum iteration limit to avoid local minima.
  • Select the preferred class solution based on a combination of BIC (primary), AIC, classification quality, interpretability, and class sizes.

Class interpretation and labeling

  • Interpret classes using item-response probabilities (probability of “yes” for each support within each class).
  • Label the three observed regimes as:
    1. high-support / comprehensive;
    2. basic STEM-only, no remediation;
    3. remediation-focused, weak STEM enrichment.
  • Present profiles graphically (e.g., line plot of item probabilities by class) to illustrate contrasts across regimes.

Associations with school characteristics

  • Model associations between school characteristics and latent class membership using multinomial logistic regression with class membership as the outcome.
  • Report relative risk ratios/odds ratios comparing each regime to a reference class (e.g., comprehensive regime).
  • If applicable, describe any weighting or design-based adjustments.

  "C1CREDREC","C1HOMEWORK","C1DROPOUT"
)

# 2. Recode: negatives -> NA, 0 -> 1, 1 -> 2
school_lca <- school %>%
  mutate(
    across(
      all_of(indicators),
      ~ case_when(
        .x < 0  ~ NA_real_,
        .x == 0 ~ 1,
        .x == 1 ~ 2,
        TRUE    ~ NA_real_
      )
    )
  ) %>%
  # keep only rows fully observed on all indicators
  filter(if_all(all_of(indicators), ~ !is.na(.)))

# 3. Define LCA formula
f_lca <- as.formula(
  paste("cbind(", paste(indicators, collapse = ", "), ") ~ 1")
)

# 4. Fit models for K = 2,3,4
set.seed(1234)
lca_results <- list()
for (K in 2:4) {
  cat("Fitting K =", K, "with", nrow(school_lca), "schools\n")
  lca_results[[paste0("K", K)]] <- poLCA(
    f_lca,
    data    = school_lca,
    nclass  = K,
    maxiter = 5000,
    nrep    = 20,
    verbose = FALSE
  )
}

get_yes_col <- function(mat) {
  cat_vals <- suppressWarnings(as.numeric(colnames(mat)))

Latent class summary

The three-class latent class model of school STEM and academic supports identified three distinct regimes.

Class 1, comprising about 54% of schools, shows high probabilities of offering after-school STEM activities (≈ 0.74) and STEM-related field trips (≈ 0.68), as well as strong availability of credit recovery programs (≈ 0.88) and moderate homework and dropout prevention supports (≈ 0.54 and ≈ 0.53). Substantively, this group represents a high-support / comprehensive regime with both STEM enrichment and academic safety nets.

Class 2, representing roughly 15% of schools, exhibits lower probabilities of after-school STEM (≈ 0.46) and mentoring (≈ 0.32), with relatively higher use of STEM-related field trips (≈ 0.72). In contrast to Class 1, these schools show essentially no credit recovery or dropout prevention programs (probabilities near 0). This pattern is consistent with a basic STEM-only, no remediation regime, where some enrichment exists but formal catch-up supports are absent.

Class 3, encompassing about 31% of schools, has very low probabilities of after-school STEM (≈ 0.14) and mentoring (≈ 0.10), and a moderate probability of STEM-related field trips (≈ 0.50). Credit recovery is relatively common (≈ 0.81), whereas homework and dropout prevention supports are less prevalent (≈ 0.33–0.47). This group reflects a remediation-focused, weak STEM enrichment regime: schools prioritize credit recovery but offer fewer STEM enrichment and other support structures.
